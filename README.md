# 🛍️ Myntra Clone

### A responsive clone of the Myntra homepage built using HTML and CSS.

This project recreates the visual design and layout of the Myntra homepage, focusing on responsive design, clean layouts, and modern UI development. It was built to strengthen my frontend fundamentals by replicating a real-world e-commerce interface.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/zufshan98/MyntraClone.git
```

Navigate to the project

```bash
cd MyntraClone
```

Open `index.html` in your preferred browser.
